# Умный консультант по подбору товаров

Умный консультант по подбору товара на основе интернет-отзывов.
## Описание

Сервис анализирует отзывы о товарах с сайта iRecommend.com, определяет их тональность (положительные, нейтральные, негативные) и предоставляет пользователю детальную статистику и общую оценку товара.

## Функциональность

- Система авторизации и регистрации пользователей
- Парсинг отзывов с сайта iRecommend.com
- Анализ тональности отзывов с использованием модели `blanchefort/rubert-base-sentiment`
- Визуализация результатов анализа (процентное соотношение, общая оценка)
- История всех проведенных анализов
- Профиль пользователя с сохранением истории

## Технологии

- **Backend**: Flask (Python)
- **База данных**: PostgreSQL
- **Анализ тональности**: Transformers (blanchefort/rubert-base-sentiment)
- **Парсинг**: BeautifulSoup4, Requests
- **Frontend**: HTML, CSS, JavaScript

## Использование

1. Зарегистрируйтесь или войдите в систему
2. Перейдите на страницу "Анализ"
3. Вставьте ссылку на товар с сайта iRecommend.com
4. Нажмите "Анализировать отзывы"
5. Просмотрите результаты анализа
6. Все анализы сохраняются в вашем профиле
