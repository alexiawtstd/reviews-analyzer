# Умный консультант по подбору товаров

Умный консультант по подбору товара на основе интернет-отзывов.
## Описание

Сервис анализирует отзывы о товарах с сайта iRecommend.ru, определяет их тональность (положительные, нейтральные, негативные) и предоставляет пользователю детальную статистику и общую оценку товара.

## Функциональность

- Система авторизации и регистрации пользователей
- Парсинг отзывов с сайта iRecommend.ru
- Анализ тональности отзывов с использованием модели `seara/rubert-tiny2-russian-sentiment`
- Визуализация результатов анализа (процентное соотношение, общая оценка)
- Профиль пользователя с сохранением истории

## Технологии

- **Backend**: Flask (Python)
- **База данных**: PostgreSQL, SQLite (для локальной разработки), Flask-SQLAlchemy
- **Анализ тональности**: torch, transformers (Hugging Face) с моделью `seara/rubert-tiny2-russian-sentiment`
- **Парсинг**: beautifulSoup4, cloudscraper, request
- **Frontend**: HTML, CSS, JavaScript
## Использование

1. Зарегистрируйтесь или войдите в систему
2. Перейдите на страницу "Анализ"
3. Вставьте ссылку на товар с сайта iRecommend.com
4. Нажмите "Анализировать отзывы"
5. Просмотрите результаты анализа
6. Все анализы сохраняются в вашем профиле
